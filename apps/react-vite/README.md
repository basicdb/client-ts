# React Vite Demo App

Demo application for validating the `@basictech/react` SDK in a plain React + Vite setup.

## Commands

```bash
npm run dev
npm run build
npm run lint
```

## Key Files

- `src/main.tsx`: `BasicProvider` bootstrap.
- `src/App.tsx`: SDK dashboard/test UI.
- `basic.config.ts`: local schema and project config.
